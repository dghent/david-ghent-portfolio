Scoot Operations

To keep Scoot's shared electric vehicle network running, there are vehicle engineers, fleet supervisors, managers and customer service reps that are constantly monitoring vehicles. Scoot's admin desktop experience allows users to do things like adjust a vehicle's firmware settings remotely, remove a vehicle from the public fleet, or create a list of vehicles in a specific state so tasks can be created to resolve them.
