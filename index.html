Scoot Operations

To keep Scoot's shared electric vehicle network running, there are vehicle engineers, fleet supervisors, managers and customer service reps that are monitoring all vehicles for issues, and need to access various data and vehicle actions at any given time. Scoot's admin operations desktop experience allows users to do things like adjust a vehicle's firmware settings remotely, remove a vehicle from the public fleet, or create a list of vehicles in a specific state (like neededing maintenance) so that tasks can be created to resolve them.
